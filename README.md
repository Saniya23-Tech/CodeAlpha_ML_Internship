# CodeAlpha ML Internship 🚀

Welcome to my **CodeAlpha AI/ML Internship Repository**. This repository contains the projects completed as part of my Machine Learning internship, showcasing practical applications of AI and data-driven problem solving.

This repository documents my journey of learning and implementing Machine Learning concepts through hands-on projects. Each project focuses on understanding the complete ML workflow, from data preprocessing and feature engineering to model development and evaluation.

## Projects

### Task 1: Credit Scoring Model
Developed a machine learning model to predict an individual's creditworthiness using financial attributes and classification techniques.

### Task 2: Speech Emotion Recognition
Built an LSTM-based deep learning model using MFCC features and the RAVDESS dataset to classify emotions from speech audio. The model was integrated with Streamlit to create an interactive application.
